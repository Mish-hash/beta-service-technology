## О проекте

    Проект собран на фремворке Laravel 8.32.1

## Запус и работа

    После клонирования проекта, нужно войти в директорию приложения и установить все зависимости, а так же сгенерировать новый ключ приложения
    ```
    composer install
    cp .env.example .env
    php artisan key:generate
    ```

    После чего в файл `.env` внести конфигурацию базы данных.

## 1 Задание

    Решение первого задания реализовано в миграции и в сидах.
    ```
    php artisan migrate
    php artisan db:seed
    ```
## 2 Задание

    По маршруту `{site}/api/curency` реализовано API который возвращает в формате JSON результат валюты для переданного `valueID` за указанный период `date` `from & to`

## 3 Задание

    После выполнения миграции и наполнения базы данных (1-е задание) у нас так-же появляется пользователь системы
    ```
    user: admin@example.com
    password: 123qwerty123
    ```
    Где мы после авторизации можем выбралюту одну или несколько, выбрать даты и получить таблицу с информацией.
